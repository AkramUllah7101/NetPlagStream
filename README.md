4. **Run the Application:**

   This command will start the services needed for plagiarism detection.

## üéØ How to Use NetPlagStream

1. **Access the Application:**
   Once the services are running, open your web browser and go to `http://localhost:8080`.

2. **Upload Your Document:**
   Click on the "Upload Document" button. Choose the file you want to check for plagiarism.

3. **Start the Detection:**
   After uploading, click on the "Check for Plagiarism" button. The application will analyze your document and provide results in a few moments.

4. **Review Results:**
   The results will indicate any detected plagiarism along with sources. You can download or save the report for your records.
